## 📊 Chart Types Available

### 1. Bar Chart (Frequency Analysis)
```
Feature: Shows count comparison
Best for: Top symptoms, location comparisons
Location: HomePage, AnalyticsPage
Style: Cyan bars on dark background
```

### 2. Line Chart (Temporal Analysis)
```
Feature: Shows trends over time
Best for: Outbreak detection, progression tracking
Location: AnalyticsPage
Style: Cyan line with dots and animations
```

### 3. Pie Chart (Distribution Analysis)
```
Feature: Shows proportions
Best for: Understanding symptom mix
Location: AnalyticsPage
Style: Multi-colored segments with labels
```

### 4. Combo Chart (Risk Assessment)
```
Feature: Shows dual metrics
Best for: Geographic risk analysis
Location: AnalyticsPage
Style: Blue and red bars for different metrics
```

## 📈 Data Flow Architecture

```
Backend API (localhost:8000)
    │
    ├─→ /api/reports ────────────┐
    │                            ▼
    │                    Process Report Data
    │                            │
    │                            ├─→ Symptom Frequency
    │                            ├─→ Temporal Trends
    │                            └─→ Location Stats
    │
    └─→ /api/symptom_counts ────┐
                                 ▼
                    Frontend React Components
                                 │
            ┌────────────────────┼────────────────────┐
            │                    │                    │
            ▼                    ▼                    ▼
    SymptomBarChart     SymptomTrendChart    SymptomPieChart
            │                    │                    │
            └────────────────────┼────────────────────┘
                                 │
                    AnalyticsPage + HomePage
                                 │
                        Display to User
```

## 🔄 Data Update Cycle

### HomePage Charts
```
User visits → Data fetches → Charts render
                              ↓
                         30 second timer
                              ↓
                         Auto-refresh data
```

### Analytics Page
```
User clicks Analytics → Load page → Fetch all data
                                      ↓
                              All 4 charts load
                              simultaneously
                                      ↓
                              Display analytics
```

## 📊 Statistics Displayed

### Summary Cards
```
Total Reports     → Count of all submissions
Unique Symptoms   → Number of different symptoms tracked
Most Common       → Top symptom by frequency
```

### Chart Metrics
```
Bar Chart      → Symptom count (0-200+)
Pie Chart      → Symptom percentage (0-100%)
Line Chart     → Daily report count (0-50+)
Location Chart → Risk index (0-10) + report count
```
